
# MySocial (frontend)

This is simple social media app using React and tailwindcss


## Run Locally

Clone the project

```bash
  git clone https://github.com/Studawatta/react-social-medial-app---frontend.git
```



Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Client:** React, TailwindCSS
